How to Set Permissions when Creating a Directory #

mkdir -m 700 newdir

Use useradd [username] to add a new user

By defualt the account will be created without a password and will be locked

Use passwd [username] to add a new user

Run password by itself to set your own password

Use groupadd [username] to ceate a new group

Use gpasswd -a [user] [group] to add a user to a group

# Docker cheat sheet ## Start Docker deamon service $ sudo service docker start ## Status of Docker deamon service $ sudo service docker status ## Stop Docker deamon service $ sudo service docker stop ## Pulling an image from docker hub $ sudo docker pull ubuntu:20.04 ## Show pulled images $ sudo docker images ## Create a container with a image $ sudo docker run -it --name my-pytest my-selenium:latest bash ## Show existing containers $ sudo docker ps -a ## Create a container and remove the container upon exit $ sudo docker run -it --name my-pytest --rm my-selenium:latest bash ## Remove all containers $ sudo docker container prune ## Building a custom image and creating a container from the custom image $ sudo docker build -t my-selenium . && sudo docker run -it --name sele-cont --rm my-selenium bash ## Remove an image $ sudo docker rmi ubuntu:latest ## Remove images with values $ sudo docker rmi $(sudo docker images -f "dangling=true" -q) --force
